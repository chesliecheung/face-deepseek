# Smart-Blog-Platform

基于 Spring Boot 与 AI 大模型的人脸识别后端智慧博客平台项目。

## ✨ 功能简介
- 人脸识别相关接口
- AI 对话交互
- 文章、分类、评论、收藏等内容管理
- 跨域配置与统一接口规范

## 🛠️ 技术栈
- Java
- Spring Boot
- Maven
- DeepSeek AI
- Git & GitHub

## 📂 项目结构
- config：跨域、Web 相关配置
- controller：接口控制层
- service：业务逻辑层
- entity：数据实体
- mapper：数据操作

## 🚀 运行方式
1. 使用 IDEA 打开项目
2. 等待 Maven 依赖加载完成
3. 运行主启动类
4. 在本地 8080 端口访问接口
