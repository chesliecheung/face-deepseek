# Smart-Blog-Platform 🧠

基于 Spring Boot + DeepSeek AI + 百度人脸识别 打造的智慧博客平台后端，实现安全认证、智能创作、内容社交一体化服务。

## ✨ 功能简介
- 👤 人脸识别认证接口：人脸采集、特征加密存储、登录比对、失败重试
- 🤖 DeepSeek AI 智能辅助：大纲生成、内容扩写 / 精简、语法纠错、风格转换
- 📝 文章全生命周期管理：富文本编辑、定时发布、自动草稿、分类 / 标签体系
- 💬 社交互动能力：嵌套评论、点赞动画、自定义收藏夹、多渠道分享
- 🧩 工程化规范：统一响应体、全局异常处理、Redis 缓存、跨域配置

## 🛠️ 技术栈
- Java
- Spring Boot
- Maven
- DeepSeek AI
- Git & GitHub

## 📂 项目结构
- config：跨域、Web 相关配置
- controller：接口控制层
- service：业务逻辑层
- entity：数据实体
- mapper：数据操作

## 🚀 运行方式
1. 使用 IDEA 打开项目
2. 等待 Maven 依赖加载完成
3. 运行主启动类
4. 在本地 8080 端口访问接口
